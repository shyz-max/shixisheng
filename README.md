# MES 零件出库申请编排工具

这是一个本地可用的流程配置工具，用来定义 MES 系统里“零件出库申请”的操作顺序。

## 打开方式

直接用浏览器打开 `index.html`。

## 能配置什么

- 流程步骤：第 1 步点哪里、第 2 步点哪里、输入什么、选择什么、等待多久。
- 定位方式：按钮文字、CSS、XPath、坐标、人工备注。
- 零件模板：零件编码、名称、数量、库位、行号或自定义选择器。
- 最终动作：配置最后点击哪个按钮，例如提交、保存、确认。
- 执行参数：浏览器类型、默认等待、失败处理、提交前确认。

## 保存与迁移

- `保存` 会把配置存在当前浏览器本机。
- `导出 JSON` 可以把配置文件发给别人或留档。
- `导入 JSON` 可以恢复以前保存的流程。

## 下一步可扩展

已经提供 Playwright 执行器模板。安装依赖后，把界面导出的 JSON 命名为 `mes-outbound-flow.json`，放在当前目录，然后运行：

```powershell
npm install
npm run run-flow
```

也可以指定其他配置文件：

```powershell
$env:MES_FLOW_CONFIG="C:\path\to\flow.json"
npm run run-flow
```
